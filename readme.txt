关于Pusic lite
1.pusic lite 的开发本事兴趣所致，本身并无详细的开发开发目的，以练习及提升自己的mac端开发能力为主；
2.pusic lite 开发中用到了不少的小技巧：
  1@：base View 的自定义Item内容的的Table View；
  2@：dock 中程序图标的隐藏；（一，通过info.plist的设置；二，通过代码的动态控制）
  3@：使用第三方的音乐播放框架进行音乐解码；（遗憾的是暂时还未进行对Ape，fl等格式的支持，还在慢慢研究ffmeng）
  4@：尝试加入课一些mac CABaseAnimation等高级动画的使用，这新动画原理掌握较易，用好则难；
  5@：程序的主入口较为简单－－以系统状态栏中图标点击进入；
  6@：增加对系统 dark／light model主题的支持；
  7@: 这是一个简化版，完全版功能更强！
